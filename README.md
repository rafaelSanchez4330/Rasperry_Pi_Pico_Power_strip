# Rasperry_Pi_Pico_Power_strip

Repositorio dedicado a proyectoo de Rasperry Pi Pico W

Con el fin de desarrollar un multicontacto inteligente con las siguientes características:
  - Poder conectarse a una red local doméstica
  - Poder controlar los contactos a través de una aplicación web
  - Apagar y prender el flujo de corriente por los contactos
